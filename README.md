# Patient Management System (Python)

A desktop application built with **Python Tkinter and SQLite** to manage patient records in a simple hospital management system.

## Features

* Add patient records
* Update patient information
* Search patient by ID
* Delete patient records
* Display all patient data
* SQLite database integration
* User-friendly GUI

## Technologies Used

* Python
* Tkinter
* SQLite3

## How to Run

1. Make sure Python is installed.

2. Run the program:

```
python "Patient Management System.py"
```

The application will automatically create the database file if it does not exist.
